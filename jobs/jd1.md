# 岗位职责

1. 主导跨境电商业务场景的AI Agent应用设计与落地，包括不限于智能客服、市场洞察、智能选品、智能广告、库存预测、定价策略自动化等。

2. 构建稳定可扩展的Agent系统架构，包括工具链设计、任务规划（Planning）、Agent Loop、工作流调度、多Agent协作等核心模块。

3. 使用Python/Java进行Agent组件开发、API 编排、模型调用、RAG 系统搭建，以及端到端工程化落地。

4. 推动Agent与实际业务深度融合，能够在不确定性场景下制定Prompt策略、技能（Tools）体系、干预机制等，使Agent在真实生产中可控、可靠。

5. 具备完整的Agent落地闭环能力：需求分析 -> 技术方案 -> 架构设计 -> 模型与工具集成 -> 验证与评估 -> 上线发布。

6. 跟踪前沿Agent技术（LangGraph、AutoGen、Swarm、ReAct、GraphRAG 等），持续进行架构优化和性能提升。

7. 参与业务算法能力建设，如推荐、预测、决策算法，与数据、产品团队协作梳理机制和指标体系。

# 任职要求

1. 本科及以上学历，计算机、人工智能、软件工程等相关专业。

2. 3年以上开发经验，并有电商/跨境电商/供应链场景的实践经验（任意方向，如选品、补货、履约、客服、物流等）。

3. 必须有实际的 AI Agent 项目落地经验，包括但不限于：

   - 在生产环境中运行的单体Agent或多Agent系统
   - 业务流程自动化（Workflow Agents）
   - 电商/供应链场景智能化应用（如预测、客服自动化、运营自动化）
   - Agent日志监控、容错机制、工具调用可靠性设计。

4. 熟练掌握Python和Java，能够独立完成Agent相关模块、服务、工具插件开发。

5. 熟悉至少一种Agent Framework：LangChain / LangGraph / AutoGen / Haystack / Transformers Agents 等。

6. 具备以下任意一种算法能力：NLP（对话管理、意图识别、RAG、召回/排序等）、推荐/预测算法、机器学习或深度学习模型工程化。

7. 熟悉常用大模型（OpenAI、DeepSeek、Qwen、Llama 等），理解 Function Calling、Tool Use、Agent Loop、Graph-based Agents 等核心机制。

8. 有扎实的软件工程能力：微服务、API 设计、并发、消息队列、向量数据库、缓存体系等。

9. 良好的沟通与跨团队协作能力，能够把技术方案推动到真实业务环境。
