# 主题5：Agent 评测体系方法论（题库 / gold set / 冻结世界 / 北极星向量）

## 一句话定位
评测的复杂度是被"对错有多难判断"一步步逼出来的——需要裁判 → 主客观分离（客观能用代码判的绝不用模型判）→ 三层裁判塔（干活的AI/判卷器/金标集）→ 题库 → gold set → 冻结世界 → 北极星向量（Precision/Recall/有效覆盖/维护成本，不合并成一个分数，防 Goodhart）。
