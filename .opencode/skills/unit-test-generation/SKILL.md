## 何时使用

代码实现后，或修复 Bug 时，需要验证函数、组件、服务、业务规则。

## 前端单元测试重点

- 组件渲染
- 用户交互
- 表单校验
- 状态变化
- API mock
- 错误态/空态/加载态

## 后端单元测试重点

- service 业务规则
- DTO/schema validation
- 权限判断
- 数据转换
- 异常分支
- 幂等逻辑

## 输出

- 测试对象
- 测试场景
- 测试文件
- 断言点
- 运行命令
- 运行结果
